## 7. 角色归属约束实际生效（对应 §6.2 / §11.1；实现 T8）

**目标**：确保归属硬约束在运行时真实生效（非仅文档声明）。

> **独占机制（已定稿：路径 B / agent team teammate 形态，对齐 T3 `design-mcp-config-shape.md` v(路径B) §1）**。
> 运行形态裁决已落地：交付物是**一支真正的 agent team（7 个平级 teammate + 共享配置）**。Claude Code 官方行为：**teammate 不应用 subagent frontmatter 的 `mcpServers`/`skills` 字段，teammate 从 project/user settings（含共享 `.mcp.json`、项目级 `.claude/skills/`）加载 MCP 与 skills**。故「内联进 frontmatter 实现物理独占」（旧路径 A）在本形态**失效作废**。改为：
> - **核心 MCP（GitHub 多仓 + Jira）全写共享 `.mcp.json`**（会话级，所有 teammate 都连接、物理上都可见）。
